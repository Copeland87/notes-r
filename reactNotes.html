React Notes Section 1.1 Components Notes : -React is a JavaScript library for building user interfaces (UIs). -It can be used as the V in MVC (Model View Controller) -React was built to solve building large applications with data that changes over time. -Components work similarly to JavaScript functions: :They generate an output every time they are called or invoked. :The components effect the Virtual DOM - An in-memory representation of real DOM elements generated by React components before any changes are made to the page. - The in-memory represantion of what will become real elements are then transfered to the actual DOM and become elements displayed in a browswer (hopefully not IE) - This process is called Diffing, virtual DOM diffing allows React to minimize changes to the DOM as a result of user actions - therefore, increasing browswer performance.
